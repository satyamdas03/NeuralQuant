# The Claude Web Design "Cheat Sheet"

### *From Basic Layouts to Production-Ready Sites*

Most people think Claude is just a chatbot, but they’re using it the old way. You’re about to turn it into a full-scale web design agency by installing the right **Skills**.

Here is the exact list of commands and skills to upgrade Claude from "average" to "Senior UI Designer."

---

### **1. The Core Setup: Claude Code**

To use these "Skills," you need to be running **Claude Code** (the terminal version of Claude). This is where the real power lives because it can actually interact with your computer and your files.

---

### **2. The "Better UI" Skill (Design Quality)**

Most AI-generated sites look the same: purple gradients and generic cards. This skill fixes that by giving Claude a "design philosophy" before it starts coding.

- **The Command:** `npx skills add anthropics/claude-code-skill-frontend-design`
- **What it does:** It teaches Claude about visual hierarchy, bold typography, and modern color palettes. It stops Claude from making "AI-looking" sites and starts making things that look like a human designer reviewed them.

---

### **3. The "Animation Library" Skill (Motion)**

If you want your site to feel alive with high-end animations (like Framer Motion or GSAP), you need this specific skill.

- **The Command:** `npx skillfish add itsimonfredlingjack/codex-dev-plugin animation-libraries`
- **What it does:** It gives Claude expert knowledge on high-performance motion, scroll-linked animations, and page transitions. It turns a static page into a dynamic experience.

---

### **4. The "Web Design Architecture" Skill (Structure)**

This is the skill that ensures your site isn't just pretty—it's functional and responsive.

- **The Command:** `npx skillfish add aviflombaum/claude-code-in-avinyc web-design`
- **What it does:** It turns Claude into a UI/UX lead. It focuses on mobile-first architecture, WCAG accessibility (making sure the site works for everyone), and Tailwind CSS best practices.

---

### **5. Your "Production-Ready" Workflow**

1. **Install the Skills:** Run the commands above in your terminal.
2. **Define the Goal:** Instead of a generic prompt, say:
    
    > *"/frontend-design build a landing page for a luxury watch brand. Use the /animation-libraries to add smooth entry transitions."*
    > 
3. **Iterate:** Use the command `/simplify` or `/debug` if you need to clean up the code.

---

**The secret isn't asking Claude to "draw" a site—it's giving it the professional skills to build one.**
